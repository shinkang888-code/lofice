# Loffice 랜딩 UI 재설계 명세서 (v2.16.0)

## 1. 목표

Polaris Office Tools·현대 SaaS 랜딩을 참고하여 **고객에게 자랑할 수 있는** 프론트 홈을 재구성한다.

| 항목 | Before | After |
|------|--------|-------|
| 문서 열기 | 히어로 하단 대형 드롭존 중앙 버튼 | **검색창 오른쪽** 인라인 CTA |
| 히어로↔작업 섹션 간격 | `pt-20` + `mt-10` + `mt-8` + `pt-14` | **총 ~40% 축소** (`pt-12`·`mt-5`·`pt-6`) |
| 아이콘 | 이모지 | **Lucide React** + 카테고리 컬러 |
| 타이포 | 시스템 폰트 | **Plus Jakarta Sans + Noto Sans KR** |
| 인터랙션 | 기본 hover | **glass 헤더·카드 lift·그리드 패턴·stagger** |

## 4. 기능 연결 (라우트)

모든 카드·버튼은 기존 `tools.ts` / `nav.ts` `href` 유지.

- 문서 열기 → `openLocalDocument()` → 형식별 뷰어/편집기
- 검색 → `filterTools()` 실시간 필터
- 인기 도구 4개 → HWP AI, PDF 편집, PPT AI, 문서 변환

## 5. 디자인 토큰

- Primary Navy `oklch(0.27 0.09 260)`
- Gold accent `oklch(0.78 0.13 80)`
- `--font-display`: Plus Jakarta Sans
- `--font-sans`: Noto Sans KR
- 카드: `rounded-2xl`, `shadow-lo-card`, hover `translate-y-[-2px]`
- 히어로: radial gold glow + subtle grid `background-image`
